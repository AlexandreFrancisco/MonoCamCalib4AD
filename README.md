# MonoCamCalib4AD
The algorithms used throughout the NAME and URL paper, are available through a Jupyter notebooks here, so one can easily experiment and run then without any further installations.
To start, you must open the [Calibration&BirdsEyeView](https://github.com/ipleiria-robotics/MonoCamCalib4AD/blob/master/Calibration&BirdsEyeView.ipynb)] file and and go to *Open in Colab* through the icon show on top.
